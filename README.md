# NodeJS_CLI

### How to run

- This is the content in **package.json**
```
{
  "name": "nodejs",
  "version": "1.0.0",
  "description": "",
  "main": "app.js",
  "scripts": {
    "test": "echo \"Error: no test specified\" && exit 1"
  },
  "keywords": [],
  "author": "",
  "license": "ISC"
}
```
- This is the command to run after opening the terminal in **Visual code**

```
node app.js
```
